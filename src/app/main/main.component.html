Правила игры

Участники являются представителями компаний, который действуют на отпределенном рынке.
У каждого участника есть месячный бюджет на продвижение, который он может распределять по инструментам ИМ.
Применение каждого инструмента приводит к тому, что показатели ИМ меняются либо в этом месяце, либо в последующие
